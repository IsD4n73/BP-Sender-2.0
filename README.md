# Buste Paga Sender 2.0

Invia email ad una mail list con allegati, corpo della mail in html, oggetto personalizzato. 

## Funzionalità
- Salvataggio credenziali
- Impostazioni
  - Prefisso Personalizzato
  - Corpo Email
  - Separatore Personalizzato
  - Tema Dark/Light
- Log attività
- Mail List 
  - Ricerca Mail List
- Invio Email
- Guida Utente
